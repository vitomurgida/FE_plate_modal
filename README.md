# FE_plate_modal
A FEniCSx code for performing a Finite Element modal analysis of thin plates

# Introduction and motivation
This code allows for a fast computation of resonance frequencies and modes of thin plates, helping in the preliminary design of thin plated structures to avoid exiting resonance frequencies.

# Input
- plate dimensions
- plate material Young's modulus
- plate material Poisson ratio
- plate material density
- number of modes to compute
- mesh size
- edges boundary conditions
